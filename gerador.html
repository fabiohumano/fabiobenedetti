GERADOR DE PÁGINAS ESTÁTICAS

Esta ferramenta cria arquivos HTML reais para cada obra sua.
Isso garante que a Capa e o Título apareçam corretamente no WhatsApp e Facebook.

Instruções

  1. Clique no botão abaixo para baixar todas as obras.
  2. O sistema vai criar um arquivo obras.zip.
  3. Baixe e extraia esse arquivo no seu computador.
  4. Você verá uma pasta chamada obras com vários arquivos .html dentro.
  5. Faça o upload dessa pasta obras para a Hostinger (na raiz do site, junto com index.html).
